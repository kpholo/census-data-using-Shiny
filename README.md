# census-data-using-Shiny
Trends in Demographics and Income

This is an interactive dashboard with Shiny to explore census data to figure out what trends in different demographics. This dashboard is created using the "adult.csv" dataset and designed to give isight to the following questions: 
1. Does education level impact the salary?
2. Impact of age and sex on salary?
3. People from which native countries are given less income in spite of having good education?
4. Does number of hours working contribute to less income?
5. Which workclass faces less income?
